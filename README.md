# inventario_tienda

# requerimientos 
- Gestion de productos

  ➡️ Registrar: se requiere un formulario para añadir un nuevo producto con campos como: nombre, categoria, precio venta y stoc inicial(cantidad exacta de productos)

  ➡️ consultar: Una tabla o lista que muestre todos los productos (inventario)

  ➡️ Actualizar: debe tener la opcion de cambiar el precio del producto y actualizar el stock

  ➡️ borrar: opcion de eliminar productos descontinuados 

  ➡️filtro por categoria: ver solo productos de un tipo ( aseo, bebidas,)

  ➡️busqueda: un Campo para escribir en nombre del producto y que se actualice en tiempo real

- sistema de stock y estado de procuto (nuevo, viejo)

  ➡️ nuevo: si el producto tiene menos de 15 dias marcar con con verde

  ➡️ antiguo: Productos que tengan más de 30 días desde su fecha de registro sin haber sido actualizados. Se marcan con  gris
  
  ➡️ alerta de stock: si el prodcto llega a menos de 5 unidades se marca con rojo
